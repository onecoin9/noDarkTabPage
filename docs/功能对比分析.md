## 🏆 各项目特点

### Constantine's Tab (我们)

**核心优势：**

- 🌟 组件可拖拽定位系统
- 🌟 时钟深度自定义（字体/颜色/字重）
- 🌟 云端同步 (Supabase + OAuth)
- 🌟 多壁纸源 (Bing + Wallhaven + Unsplash)
- 🌟 Custom CSS 完全自定义
- 🌟 丰富的小组件系统

### TabMark

**核心优势：**

- 🔖 浏览器原生书签集成
- 🤖 AI 搜索引擎聚合（豆包/Kimi/秘塔/ChatGPT）
- 🔍 对比搜索（多引擎同时搜索）
- 📱 侧边栏 + 悬浮球（任意网页可用）
- 🔗 浏览器功能快捷方式

### Tomato Tabs

**核心优势：**

- 📝 Notion 集成
- ✅ 待办事项
- 🖼️ Bing 壁纸
